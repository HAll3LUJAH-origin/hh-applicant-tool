# HH Applicant Tool (fork)

Это форк проекта [hh-applicant-tool](https://github.com/s3rgeym/hh-applicant-tool) от [@s3rgeym](https://github.com/s3rgeym) — консольной утилиты для автоматической рассылки откликов на вакансии hh.ru.

Все права на оригинальный проект принадлежат автору. В этом форке — только технические правки (часть из них предложена автору через Pull Request) и личная конфигурация для собственного использования через Docker.

Актуальная документация, лицензия и условия использования — в [оригинальном репозитории](https://github.com/s3rgeym/hh-applicant-tool).
